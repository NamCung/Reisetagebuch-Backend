# Reiseplaner-Backend
Reiseplaner, in welchem man Standpunkte eines Ortes auf einer Karte speichern kann.

## Technologien
- Java
- Spring Boot
- Gradle

## Voraussetzungen
- Java 17+
- Gradle

## Installation & Starten

1. Repository klonen:
   git clone <repository-url>

2. Projekt bauen:
   ./gradlew build

3. Anwendung starten:
   ./gradlew bootRun
